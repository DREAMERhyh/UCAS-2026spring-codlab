# UCAS-2026spring-codlab
仓库地址：git@github.com:DREAMERhyh/UCAS-2026spring-codlab

## 仓库简介
本仓库为2026春季国科大CodLab计算机组成原理研讨课个人实验归档，仅留存各实验Verilog硬件源码与对应实验报告和课程教学课件，仅供给后续修读本课的学弟学妹查阅参考。

## 测试说明
课程依托GitLab自动化FPGA流水线完成上板校验，本仓库内所有实验代码均已通过平台流水线测试；实验4.5未开展设计，无对应代码与报告。

## 包含实验
1. PRJ1：寄存器堆、ALU基础功能部件
2. PRJ2：MIPS单周期/多周期处理器
3. PRJ3：真实内存、UART、性能计数器外设
4. PRJ4：流水线CPU、Cache、DNN加速器、DMA引擎（不含4.5）

## 参考提示
本资料仅用于学习设计思路，严禁直接照搬代码、报告提交作业，抄袭属于学术不端行为。
